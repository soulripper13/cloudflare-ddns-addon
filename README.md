## Add-ons
- **Cloudflare DDNS**: Dynamic DNS client for Cloudflare supporting IPv4 and IPv6.
## Installation
1. Go to your Home Assistant instance.
2. Navigate to **Settings** -> **Add-ons**.
3. Click the **Add-on Store** button in the bottom right.
4. Click the three dots (hamburger menu) in the top right and select **Repositories**.
5. Add the following URL: `https://github.com/soulripper13/cloudflare-ddns-addon`
6. Click **Add**, then close the dialog.
7. The "Cloudflare DDNS" add-on should now appear in the store.
